Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
2
1

В функции `test` переменная х равна единице, затем в defer выполняется  инкрементирование x и его возврат.
В функции `anotherTest` defer инкрементирует копию x,что никак не влияет на саму переменную, поэтому в результате выполнения х = 1. 

Defer's это ключевое слово для отложенного выполнения функции, пока не завершится окружающая функция. То есть, аргументы defer высчитываются сразу, но функция выполняется после завершения окружающей. Если в одной функции определено несколько defer'ов они помещаются в стек типа LIFO.
```
